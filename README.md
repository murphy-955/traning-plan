## 使用说明

使用`paln`代码块简化书写

格式如下：

```
2026-01-01 ->日期
调整 -> 训练目的
10:00-11:30 -> 训练时间(24小时制)
8×100 自由泳 放松拉长
8×100 主项 50打退/50划手 注意技术动作 -> 距离 泳姿 训练方式 注意事项 允许有空，但一定要以空格分隔
8×100 自由泳 放松拉长
8×100 主项 50主项/50拉长 注意技术动作
```

## 参考模板(Intelij IDEA)

1. 打开`Settings` -> `Editor` -> `Code Style` -> `Markdown`
2. 点击`Edit`按钮
3. 点击`Templates`标签
4. 选择`Plan`模板
5. 点击`Apply`按钮

| 变量名         | 表达式                                                                                                                                                                              | 默认值 | 说明                                      |
|-------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----|-----------------------------------------|
| `plan-date` | `enum("周一", "周二", "周三", "周四", "周五", "周六", "周日")`                                                                                                                                 | /   | 周一到周日选一个                                |
| `plan-type` | `enum("有氧", "高有氧", "力量", "速度", "比赛速度", "调整", "陆上")`                                                                                                                              | /   | 7种训练目的                                  |
