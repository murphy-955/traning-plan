## 项目介绍

最近在给我们网站做开发，然后领导说我们网站有些单调了，建议我可以分享一下我的训练计划。我刚好也有这个想法， 于是就有了这个项目。

当然这个项目也有很大的程度上是为了方便我们大学的游泳协会的成员，在我毕业之后，就算没有人来管理训练，也不会感到迷茫，不知道自己要怎么练，练什么。此外，也希望能帮助到社会游泳爱好者，让各位有个参考。

我勉强算是一个有半吊子水平的游泳爱好者。很幸运，现实暂时还没有摩灭我对游泳的热情。在我的热情消耗干净之前，我还是想把我的训练计划分享给大家。如果没有意外的话，我会每周日更新这一周的训练计划，希望大家能够从中受益，一起进步。
本人是计算机科学与技术专业的学生，对体育训练与恢复方面不甚了解，所以下面分享的训练计划仅供大家参考。另外，也希望有这方面的大佬能提供一些建议，可以开一个Issue或者PR，一起讨论(
⊙o⊙)~

当然啦，后续我也会将我的训练计划分享到我们协会的网站上，至于要不要搞国际化还是要看实际情况，大家敬请期待~^~
